# TechStore — Интернет-магазин электроники

Учебный проект по дисциплине **«Фронтенд и бэкенд разработка»** (4 семестр, 2025/2026).  
Реализован в рамках Практических занятий 1–5.

## Стек технологий

- **Бэкенд:** Node.js, Express, nanoid, cors, swagger-jsdoc, swagger-ui-express
- **Фронтенд:** React, Axios, Sass

## Структура проекта

```
project/
├── server/       # Express API
│   └── app.js
└── client/       # React приложение
    └── src/
        ├── api/
        ├── components/
        └── pages/
```

## Функциональность

- Просмотр списка товаров с фильтрацией по категориям
- Создание, редактирование и удаление товаров
- 11 предустановленных товаров (смартфоны, ноутбуки, наушники и др.)
- Карточка товара: название, категория, описание, цена, остаток на складе, рейтинг
- Swagger-документация с интерактивным тестированием

## Запуск проекта

### Бэкенд

```bash
cd server
npm install
node app.js
```

Сервер запустится на `http://localhost:3000`  
Swagger UI доступен по адресу `http://localhost:3000/api-docs`

### Фронтенд

```bash
cd client
npm install
npm run dev
```

Приложение откроется на `http://localhost:3001`

## API эндпоинты

| Метод | Путь | Описание |
|---|---|---|
| GET | /api/products | Список всех товаров |
| GET | /api/products/:id | Товар по ID |
| POST | /api/products | Создать товар |
| PATCH | /api/products/:id | Обновить товар |
| DELETE | /api/products/:id | Удалить товар |

## 1 Практика

фотография) с применением переменных, миксина и
использованием вложенной структуры селекторов SASS

## 2 Практика

Я реализовал API, которое предоставляет CRUD операции для списка
товаров (просмотр всех товаров, просмотр товара по id, добавление товара,
редактирование товара, удаление товара). Объект товара содержал
следующие поля: id, название, стоимость.

## 3 Практика

1. Протестировал  API из Практического занятия 2 с помощью
Postman (не менее 3-х запросов).
2. Выберал API и получил ключ. Изучил документацию и
выполнил не менее 5-ти запросов.

## 4 Практика

Доработал рассмотренный пример: два небольших приложение на React и
Express (для клиента и сервера, соответственно), связанных в единую систему - сайт
интернет-магазина товаров с тематикой на ваше усмотрение. Количество товаров - не
менее 10. Карточка товара содержит: название, категорию,
описание товара, цену, количество на складе. Опционально доработать рейтинг и фото.

## 5 Практика

Доработал задание из практического занятия №4: подключил к нему
swagger-jsdoc и swagger-ui-express, опишисал с помощью JSDoc-аннотаций схему
пользователя ( User) и все CRUD-операции (GET , POST , GET/:id, PATCH/:id,DELETE ).
Убедился, что документация доступна по адресу /api-docs и работает в
интерактивном режиме (можно отправить тестовый запрос).
